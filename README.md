# App

## Setup
Please setup environment variable before running
- for Windows
```set POSTGRES_USER=postgres```
```set POSTGRES_PASSWORD=postgres```

## Backend
### Installation
* install python (^3.10)
* install requirements.txt
### Run
* uvicorn main:app --reload
(set port if needed)

## FrontEnd
### Installation
* install nodejs
* install vue
* Execute: `npm install`
### Run
* Execute: `npm run serve`
